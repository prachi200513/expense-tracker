# 💰 Expense Tracker

A simple REST API built using Spring Boot to manage daily expenses.

This application allows users to:
- ➕ Add new expenses
- 📋 View all expenses
- ❌ Delete expenses

---

## 🚀 Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Gradle
- H2 / MySQL
- REST API
